# NFT Marketplace

Blockchain-based NFT marketplace project developed as a team project using React.js, Solidity, Hardhat, and IPFS.

## Features
- MetaMask wallet integration
- NFT upload using IPFS (Pinata)
- Frontend built with React.js and Tailwind CSS
- Smart contract integration support

## Technologies Used
- React.js
- Tailwind CSS
- Solidity
- Hardhat
- MetaMask
- IPFS (Pinata)

## Project Type
Academic Team Project
